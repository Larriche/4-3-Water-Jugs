This is a Java program to print out the solution to the 
4-3 Water jugs puzzle described below:

You are given two (2) jars, a four-gallon one and a three-gallon one. Neither has any measuring markers on it. There is a tap that can be used to fill the jars with water. How can you get exactly two gallons of water into the four-gallon jar? 

The solution uses breadth first search as described in Chapter 3 of Artificial Intelligence - A modern approach

The main program is in Solution.java.
